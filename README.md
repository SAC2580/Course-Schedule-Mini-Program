基于微信小程序的课表查询系统的设计与实现

技术说明：
用户前端：微信小程序原生框架
管理前端：Vue.js
服务端：基于Java的SpringBoot框架
数据库：MySQL

功能说明：
用户端：
1. 首页
    展示Banner轮播图
    公告滚动条
    快捷功能入口宫格
    校园资讯列表
2. 课程表
    支持按周次切换
    日视图查看课程安排
    按院系/班级筛选
    课程卡片按颜色区分
3. 成绩查询
    按学期分组查看成绩
    统计平均GPA/总学分
    柱状图
    饼图
    折线图可视化
4. 考试查询
    考试列表展示
    倒计时计算
    关键词搜索
5. 空教室查询
    按周次、星期、节次、楼栋多条件查询空闲教室
6. 课程列表
    全部课程展示
    关键词搜索
7. 课程详情
    课程基本信息
    排课时间
    课程评价（含提交评论）
    收藏
    取消收藏
8. 校园资讯
    资讯列表展示
    下拉刷新
9. 资讯详情
    富文本渲染
    图片路径自动补全
    图片预览
10. 公告通知
    Tab分类展示（全类型分类）
    展开/收起详情
11. 教师信息
    教师列表展示
    按姓名/工号/学院关键词搜索
12. 个人中心
    用户信息展示
    功能菜单导航（编辑资料/收藏/课表/成绩/考试/公告）
13. 编辑资料
    头像上传与修改
    昵称/姓名/学院/专业/班级/年级/手机/性别编辑
    密码修改
14. 登录
    学号加密码登录
    默认密码123456
15. 我的收藏
    收藏课程列表展示
    取消收藏
    点击跳转课程详情

管理端：
1. 首页统计分析
2. 轮播图片管理
3. 系统公告管理
4. 校园资讯管理
5. 学期信息管理
6. 楼栋信息管理
7. 教室信息管理
8. 教师信息管理
9. 课程信息管理
10. 课程安排管理
11. 考试信息管理
12. 成绩信息管理
13. 用户信息管理
14. 课程评价管理
15. 节次时间管理
16. 管理账号管理

项目创新点：
1. 课程表支持按周次切换、日视图查看及颜色区分，直观展示课程安排
2. 空教室多条件查询（周次/星期/节次/楼栋），提高自习空间利用率
3. 成绩查询集成平均GPA与总学分统计，结合柱状图/饼图/折线图多维度可视化
4. 考试倒计时计算与关键词搜索，帮助学生高效管理备考计划
5. 教师信息按姓名/工号/学院多维度搜索，便于师生联系与了解
6. 课程评价与收藏联动，支持学生分享课程体验与构建个人课单
7. 公告通知Tab分类展示，支持展开/收起详情，信息获取更高效
